# WhiteJonas.github.io

个人主页 / 作品集，托管于 GitHub Pages：**https://whitejonas.github.io**

- 纯静态单页（`index.html` + `style.css`），零构建、零依赖
- 极简技术风，支持深色模式，响应式适配手机
- 内容：研究方向（LLM 推理调度）· 项目 · 技能

## 本地预览

直接双击 `index.html`，或用任意静态服务器：

```bash
python -m http.server 8000
# 打开 http://localhost:8000
```

## 修改内容

所有内容都在 `index.html` 里，直接改文字即可。需要更新的占位符：

- 邮箱：搜索 `your-email@example.com` 全部替换成你的真实邮箱
- 项目：每个 `<article class="proj">` 是一个项目卡片，可增删
